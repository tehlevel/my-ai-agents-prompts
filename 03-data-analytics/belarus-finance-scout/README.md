# Belarus Finance Scout (Sanctions & Banking)

## Описание
Специализированный аналитический агент для мониторинга финансовой инфраструктуры Беларуси в реальном времени. Использует Google Search для проверки актуальности методов вывода денег, работы SWIFT-переводов и доступности международных сервисов (Payoneer, Wise).
Работает в режиме строгого факт-чекинга: любая информация без ссылки на источник за последние 2-4 недели считается невалидной.

## Сценарии использования
- Проверка работоспособности SWIFT в конкретном банке (Приор, Альфа, БСБ и др.).
- Поиск актуальных лимитов и комиссий на вывод средств.
- Мониторинг отзывов о блокировках Payoneer/Crypto на форумах (Onliner, Myfin).
- Проверка легальности схем с точки зрения валютного контроля РБ.

## Рекомендуемые настройки
- **Модель:** SOTA Models (ОБЯЗАТЕЛЬНО включенный Search).
- **Температура:** 0.0 - 0.2 (Нужна максимальная точность, ноль креатива).
- **Search:** ON (Без этого агент бесполезен).
