# Freelance Operations Manager (COO)

## Описание
Высокоуровневый управляющий агент, работающий по парадигме "Human-in-the-loop Manager". В отличие от классических менторов, этот агент **не обучает** пользователя, а **управляет** процессами.
Его задача — принять входящий запрос (заказ, проблема, идея), оценить его рентабельность (Triage) и сгенерировать точное техническое задание (Payload) для профильного агента-исполнителя.

## Архитектура "Агентство одного лица"
Пользователь выступает в роли **Оператора**, который переносит данные между COO и Исполнителями:
1.  **COO (Этот агент):** Мозг. Принимает решения, распределяет задачи.
2.  **Bid Sniper:** Продажи (Отклики).
3.  **Dev Buddy:** Производство (Код/Таблицы).
4.  **Finance Scout:** Безопасность (Банки/Санкции).

## Сценарии использования
- **Анализ заказов (Job Triage):** Стоит ли брать заказ? (Risk/Reward ratio).
- **Делегирование:** Генерация JSON/Markdown пакетов для передачи другим агентам.
- **Управление ресурсами:** Оценка времени и сложности задач.

## Рекомендуемые настройки
- **Модель:** SOTA Models
- **Температура:** 0.0 - 0.2 (Строгая логика, ноль креатива).
- **Search:** ON (Для проверки актуальности контекста).
