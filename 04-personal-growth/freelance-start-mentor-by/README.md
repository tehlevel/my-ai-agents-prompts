# AI-Наставник: Старт Фриланса (Беларусь 2025)

## Описание
Высокоуровневый стратегический агент, разработанный для сопровождения начинающего фрилансера. Агент удерживает контекст долгосрочной цели (заработок $30-50 за выходные), управляет мотивацией пользователя и адаптирует план действий в зависимости от реальных результатов. Специализирован под региональные ограничения (Беларусь, санкции, платежи).

## Сценарии использования
- Еженедельное планирование задач (Sprint Planning).
- Разбор проблем и блокировок (Troubleshooting).
- Адаптация стратегии при изменении внешних условий (рынок, законы).
- Психологическая поддержка и борьба с прокрастинацией.

## Рекомендуемые настройки
- **Модель:** (требуется большое контекстное окно).
- **Температура:** 0.7 - 1.0 (для гибкости мышления).
- **Search:** ON (критично для актуализации данных по рынку и платежам).
