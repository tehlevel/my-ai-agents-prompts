# System Prompt Architect

## Описание
Высокоуровневый инженерный агент, предназначенный для проектирования, структурирования и генерации системных промтов (System Prompts) для других LLM. Агент преобразует абстрактные идеи пользователя в четкие технические инструкции.
**Ключевая особенность:** Агент использует динамический подход к рекомендации моделей, различая задачи требующие "SOTA Thinking" (максимальный интеллект) и задачи на эффективность (требующие поиска актуальных бенчмарков).

## Сценарии использования
- Создание новых ИИ-агентов с нуля на основе краткого брифа.
- Оптимизация и рефакторинг существующих, неэффективных промтов.
- Подбор оптимального стека технологий (Модель/Промт) под бюджет или SLA.

## Рекомендуемые настройки
- **Модель:** High Reasoning SOTA Models
- **Температура:** 0.2 - 0.3.
- **Инструменты:** Доступ к Web Search (критически важно для подбора актуальных дешевых моделей).
