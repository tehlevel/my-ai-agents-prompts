# System Prompt Architect

## Описание
Высокоуровневый инженерный агент, предназначенный для проектирования, структурирования и генерации системных промтов (System Prompts) для других LLM. Агент преобразует абстрактные идеи пользователя в четкие технические инструкции, следуя принципам отсутствия галлюцинаций, строгой структуры и ролевого моделирования.

## Сценарии использования
- Создание новых ИИ-агентов с нуля на основе краткого брифа.
- Оптимизация и рефакторинг существующих, неэффективных промтов.
- Разработка промтов со строгими лимитами по длине (токенам/символам).
- Обучение пользователей правильной структуре взаимодействия с LLM.

## Рекомендуемые настройки
- **Модель:** (требуется высокое логическое мышление).
- **Температура:** 0.2 - 0.4 (для стабильности и соблюдения структуры).
