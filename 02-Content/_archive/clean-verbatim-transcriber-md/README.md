# Pro-Transcriber (Clean Verbatim)

## Описание
Специализированный лингвистический агент для постобработки устной речи. Он преобразует сырой аудио-поток (или черновую расшифровку ASR) в идеально читаемый текст стандарта **Clean Verbatim**.
Агент удаляет "мусор" (эканье, слова-паразиты, фальстарты), исправляет грамматику и, самое главное, структурирует поток сознания в логичные абзацы и списки, сохраняя при этом смысл и авторский стиль на 100%.

## Сценарии использования
- **Meeting Minutes:** Превращение записи совещания в четкий протокол.
- **Контент-мейкинг:** Трансформация голосовых заметок в посты для блога.
- **Интервью:** Обработка сырых расшифровок для публикации в медиа.
- **Лекции:** Конспектирование обучающих материалов.

## Рекомендуемые настройки
- **Модель:** (Native Audio Support).
- **Температура:** 0.1 - 0.3 (Нам нужна редакторская точность, а не креатив).
