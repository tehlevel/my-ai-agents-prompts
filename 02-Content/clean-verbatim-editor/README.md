# Clean Verbatim Editor

## Описание (Role & Objective)
Специализированный агент для обработки транскрибаций и черновиков устной речи. Преобразует хаотичный речевой поток (аудио или сырой текст) в литературный стандарт **Clean Verbatim**. Агент удаляет "мусор", исправляет грамматику и структурирует текст, сохраняя 100% фактологической точности и оригинального смысла.

## Ключевые особенности (Key Features)
- **Умная фильтрация:** Удаление слов-паразитов, фальстартов и самоисправлений.
- **Грамматическая реконструкция:** Исправление согласований и синтаксиса без изменения стиля спикера.
- **Визуальное структурирование:** Автоматическое разбиение на абзацы, списки и заголовки.
- **Обработка спецтерминов:** Корректное написание брендов и технической лексики (CamelCase, верный регистр).
- **Условное саммари:** Генерирует блок "Главное" только для объемных текстов (>150 слов).

## Рекомендуемые параметры (Configuration)
*Основано на архетипе: Type 2 (Data Extraction & Formatting) с элементами Type 1.*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | 0.2 - 0.3 | Низкая температура критична для сохранения фактической точности и стабильности формата. |
| **Thinking Level**| Low | Агенту нужно минимальное время на "размышление" для проверки омонимов и контекста, но глубокие рассуждения излишни. |
| **Google Search** | OFF | Работа ведется строго с предоставленным контекстом. |
| **URL Context** | ON | Разрешено чтение ссылок, если пользователь предоставляет их как источник для транскрибации. |
