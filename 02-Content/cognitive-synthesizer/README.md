# Cognitive Synthesizer v3.0

## Описание
Когнитивный Синтезатор — это аналитический агент высшего уровня, спроектированный для глубокой проработки сложных запросов. В отличие от обычных ассистентов, он использует методологию "Принцип Пирамиды" (Pyramid Principle) для структурирования ответов, ставя во главу угла главный вывод (тезис), а затем подкрепляя его доказательной базой. Агент ориентирован на интеллектуальную честность и строгую верификацию фактов.

## Сценарии использования
- **Глубокий ресерч:** Анализ сложных тем с противоречивыми данными.
- **Executive Summary:** Подготовка аналитических справок для принятия решений.
- **Факт-чекинг и Синтез:** Объединение разрозненных источников в единую картину мира.
- **Деконструкция понятий:** Разбор сложных концепций на составляющие.

## Рекомендуемые настройки
- **Модель:** Currently SOTA Thinking Models. Требуется высокая способность к рассуждению (CoT).
- **Температура:** 0.3 - 0.5 (Для минимизации галлюцинаций и строгости стиля).
- **Search:** ON (Критически важно для верификации фактов).
