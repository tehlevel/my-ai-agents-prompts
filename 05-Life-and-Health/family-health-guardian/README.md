# Family Health Guardian

## Описание
Персональный медицинский ассистент для всей семьи (Мама, Папа, Сын). Агент анализирует симптомы, ведет историю здоровья и дает рекомендации, строго основанные на принципах Доказательной Медицины (EBM/DocMed).

**ВАЖНО:** Агент не ставит диагнозы. Он помогает структурировать данные для врача и отсеивает ненаучные методы лечения.

## Ключевые особенности
- **Multi-Profile Support:** Автоматически переключает протоколы между педиатрией (AAP/WHO) и взрослой терапией (UpToDate/NHS).
- **Red Flag System:** Мгновенное распознавание опасных состояний (Tier 1).
- **Anamnesis Bridge:** Генерация профессиональной сводки для передачи лечащему врачу (`/doctor_summary`).

## Рекомендуемые параметры
*Основано на архетипе: Type 4 (Verified Research)*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | 0.3 - 0.5 | Низкая температура для минимизации галлюцинаций в медицинских фактах. |
| **Thinking Level**| High | Требуется глубокий анализ симптомов и исключение ложных корреляций. |
| **Google Search** | ON | Для проверки лекарственных взаимодействий и локальных эпидемиологических данных. |
| **URL Context** | ON | Для чтения загруженных PDF с анализами. |

## Настройка (Конфигурация)
Перед использованием откройте `SYSTEM_PROMPT.md` и заполните раздел **FAMILY REGISTRY**:
1. Впишите даты рождения.
2. Укажите известные хронические особенности или аллергии.
