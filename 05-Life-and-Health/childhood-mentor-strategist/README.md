# Childhood Mentor Strategist

## Описание
Центральный агент-консультант ("Hub") для вопросов воспитания, психологии и детского развития. Спроектирован как долгосрочный партнер для семьи с ребенком дошкольного возраста (около 5 лет на данный момент).
В отличие от стандартного чата, агент удерживает "Конституцию семьи": гуманистические ценности (Зицер, Петрановская) и научные стандарты безопасности (ВОЗ, AAP).

## Ключевые особенности
- **Ценностный фильтр:** Автоматически отсекает вредные советы (физические наказания, шейминг, "дать сдачи"), заменяя их на научно обоснованные стратегии.
- **Context Branching (Ветвление):** Умеет генерировать "брифы" (сжатый контекст) для создания новых чатов при решении сложных, узких задач.
- **Динамический возраст:** Рассчитывает точный возраст ребенка на основе даты рождения.

## Рекомендуемые параметры (Configuration)
*Архетип: Type 1 (Complex Logic) + Type 4 (Verified Research)*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | `0.7 - 1.0` | Нужна гибкость для подбора психологических скриптов, но строгость в вопросах безопасности. |
| **Thinking Level**| `High` | Критически важно для анализа скрытых причин поведения ребенка (Why acts vs How looks). |
| **Google Search** | `ON` | Для проверки актуальных рекомендаций (вакцины, эпидемии, новые исследования). |

## Инструкция по использованию

### 1. Повседневный режим (Hub)
Используйте этот чат для фиксации событий и быстрых вопросов:
> "Сегодня отказался есть суп, хотя раньше любил. Это нормально?"
> "Поссорились из-за мультика, я сорвался. Как помириться?"

### 2. Режим "Глубокая проработка" (Branching)
Если проблема сложная (например, хроническая агрессия или страхи), не пытайтесь решить её в длинном чате.
1. Напишите команду: `/branch` или *"Сделай бриф для новой сессии по этой проблеме"*
2. Агент выдаст блок кода (текст).
3. Скопируйте этот текст.
4. Откройте **новый чат** (New Chat) с тем же агентом.
5. Вставьте текст.
6. Получите глубокий разбор без "шума" старой переписки.
