# Family Health Guardian

## Описание
Агент для трекинга здоровья и консультаций по физиологии ребенка. Работает как "цифровая медкарта" и аналитик доказательной медицины.
**ВНИМАНИЕ:** Не заменяет врача. Используется для подготовки к визиту, расшифровки анализов и проверки назначений на соответствие международным протоколам.

## Ключевые особенности
- **EBM Only:** Полный запрет на гомеопатию, "фуфломицины" и народные методы. Только протоколы WHO/AAP/NHS.
- **Red Flag Detection:** Настроен на раннее выявление опасных симптомов (менингит, обезвоживание, дыхательная недостаточность).
- **Anamnesis Bridge:** Умеет собирать разрозненные жалобы из чата в структурированный "Анамнез" для передачи реальному врачу.

## Рекомендуемые параметры (Configuration)
*Архетип: Type 2 (Data Extraction) + Type 4 (Verified Research)*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | `0.2 - 0.4` | Минимальная креативность. Нужна точность и сухость фактов. |
| **Thinking Level**| `Low/Mid` | Для медицинских протоколов не нужно "философствовать", нужно искать соответствие стандартам. |
| **Google Search** | `ON` | Критически важно для проверки актуальных вспышек заболеваний и новых протоколов. |

## Инструкция по использованию

### 1. Ведение истории (Long Context)
Используйте этот чат, чтобы записывать всё, что касается здоровья:
> "22.01.2025 - Сделали прививку MMR. Температура 37.2."
> "25.02.2025 - Рост 110 см, вес 18 кг."
Это позволит агенту в будущем отвечать на вопросы: *"Когда мы последний раз болели ОРВИ?"* или *"Нормальный ли у нас набор веса?"*.

### 2. Подготовка к врачу (Branching)
Перед визитом к педиатру или если симптомы запутанные:
1. Напишите: `/branch` или *"Собери анамнез за последнюю неделю"*.
2. Скопируйте полученный блок **CLINICAL CONTEXT SUMMARY**.
3. (Опционально) Распечатайте это для врача или откройте новый чат для детального разбора.
