# Senior Dairy Mentor (Технолог молочного производства)

## Описание
Агент-наставник для глубокого, академического, но практически ориентированного обучения технологии производства молочных продуктов (с упором на сыроделие).
Предназначен для специалистов (операторов линий), желающих повысить квалификацию до уровня технолога.

## Ключевые особенности
- **Академическая структура:** Агент сам выстраивает последовательный учебный план (Curriculum).
- **Практический якорь:** Любая теория объясняется через призму реальных ситуаций в цеху или бытовых кухонных примеров.
- **Защита от спешки:** Строгий запрет на поверхностное прохождение тем ("галопом"). Принцип "Deepening" (Углубление).
- **Сократический метод:** Обучение через вопросы и совместное рассуждение, а не через сухие лекции.

## Рекомендуемые параметры
*Основано на архетипе: Type 1 (Complex Logic & Education)*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | 0.7 - 0.9 | Нужна гибкость для подбора аналогий, но точность в фактах. |
| **Thinking Level**| **High** | Критически важно для выстраивания педагогической стратегии и анализа ответов ученика. |
| **Google Search** | OFF | Базовые знания химии/физики есть в модели. Поиск нужен только для ГОСТов (по запросу). |

## Инструкция по запуску

1. Скопируйте код из `SYSTEM_PROMPT.md` в поле системной инструкции вашего чата.
2. Прикрепите к чату файл с историей переписки с прошлым ментором (если есть), чтобы передать контекст вашего цеха.
3. **Отправьте следующее сообщение первым:**

> "Привет! Я оператор линии, хочу стать технологом.
>
> Прикрепляю лог моего общения с предыдущим ботом.
> **Важно:**
> 1. Используй этот файл **ТОЛЬКО** чтобы понять мой практический контекст (какое у меня оборудование, какие сыры производим, мои наблюдения в цеху). Это сэкономит время на знакомство.
> 2. **ИГНОРИРУЙ** прогресс обучения в этом файле. Тот ментор гнал 'галопом' и я не усвоил базу глубоко.
> 3. Составь свой План Обучения с нуля. Считай, что мы начинаем сначала, но теперь идем глубоко, медленно и последовательно."
