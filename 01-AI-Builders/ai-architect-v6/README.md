# AI Architect v6.0

## Описание (Role & Objective)
Элитный мета-агент для проектирования, аудита и создания профессиональных системных инструкций (System Prompts) в формате XML. Действует как эксперт-консультант: вместо немедленной генерации кода, он сначала анализирует задачу, задает уточняющие вопросы и согласовывает структуру. Генерирует финальные артефакты только после явного подтверждения пользователя ("Зеленый свет"). Оптимизирован для экосистемы Vertex AI и специфики моделей Gemini 1.5/2.0/3.0.

## Ключевые особенности (Key Features)
- **Консультативный протокол:** Строгий запрет на "ложный старт". Агент не выдает код, пока полностью не поймет задачу и не получит подтверждение.
- **Матрица когнитивной нагрузки:** Использует новую парадигму настройки параметров (High Temp + Reasoning) для сложных задач, отказываясь от устаревших линейных правил.
- **Интеграция с репозиторием:** Знает файловую структуру пользователя (01-05) и автоматически определяет место для нового агента.
- **Двойной артефакт:** Вместе с кодом промпта всегда создает стандартизированный файл документации.
- **Зеркалирование языка:** Адаптирует контент под язык пользователя, сохраняя технические определения на английском (XML-native).

## Рекомендуемые параметры (Configuration)
*Основано на архетипе: 1. Complex Logic & Coding*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | 0.8 - 1.0 | Для задач архитектуры необходима высокая температура, чтобы режим Reasoning (System 2) мог эффективно перебирать гипотезы и не "застревать" в шаблонных решениях. |
| **Thinking Level**| High | Критически важен. Позволяет модели провести симуляцию работы создаваемого промпта и проверить его на логические петли перед генерацией. |
| **Google Search** | OFF | Агент опирается на встроенные знания о prompt engineering и XML, внешний поиск может внести шум. |
| **URL Context** | OFF | По умолчанию не требуется, если только не загружается специфическая документация проекта. |
