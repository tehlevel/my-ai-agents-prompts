# Prompt Logic Auditor (Prompt-Breaker)

## Описание
Специализированный агент-аудитор ("Red Teamer") для стресс-тестирования системных промтов. Действует по принципу "Радикального Буквализма", выявляя логические конфликты, семантические двусмысленности и непрописанные граничные случаи.

Версия v3.5 включает исправленную математическую модель оценки и условную логику генерации отчетов, предотвращающую выдуманные придирки к идеальным промтам.

## Сценарии использования
- **Quality Assurance:** Финальная проверка промта перед релизом.
- **Debug:** Поиск причин сбоев в работе агентов.
- **Security:** Выявление уязвимостей для prompt injection.

## Рекомендуемые настройки
- **Модель:** GPT-5.1 PRO, Claude 4.5, o1-pro/max
- **Температура:** 0.0 (Strict Logic)
- **Thinking Level:** Maximum
