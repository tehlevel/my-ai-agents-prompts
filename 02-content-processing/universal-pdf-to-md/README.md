# Universal PDF to Markdown Converter (Interactive)

## Описание
Универсальный агент для оцифровки документов любой тематики (финансы, наука, лайфстайл). Ключевая особенность — режим **"Тандем"**: агент сначала анализирует визуальную структуру документа, определяет сложные элементы (таблицы, формулы, многоколоночная верстка) и согласовывает с пользователем стратегию конвертации перед началом работы.

## Сценарии использования
- **Финансовые отчеты:** Точный перенос сложных таблиц.
- **Научные статьи:** Сохранение иерархии заголовков, сносок и формул.
- **Журнальная верстка:** Адаптация многоколоночного текста и врезок в линейный Markdown.

## Рекомендуемые настройки
- **Модель:** (Обязательно с поддержкой Vision/Multimodal).
- **Температура:** 0.0 - 0.3 (Максимальная точность).

Архитектурное решение:
    1.  **Фаза Калибровки:** Это главное отличие от обычных конвертеров. Агент намеренно останавливается (Stop Sequence в логике поведения) после анализа файла, чтобы спросить пользователя о стратегии. Это решает проблему "универсальности": для кулинарии мы выберем один стиль, для балансового отчета — другой.
    2.  **Vision-First Approach:** В промте явно указано "ориентируйся на визуальное расположение". Это критично для PDF, где программный слой текста часто идет в неверном порядке (например, сначала левая колонка, потом правая, или вперемешку).
    3.  **Обработка ошибок:** Добавлен блок "Отчет о конвертации", чтобы агент честно признавался, где он мог накосячить (например, в неразборчивом скане).
