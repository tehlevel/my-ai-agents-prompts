# Freelance Bid Sniper (Upwork/Kwork)

## Описание
Специализированный агент-копирайтер для генерации высококонверсионных сопроводительных писем (Cover Letters). Использует техники "Mirroring" (отзеркаливание лексики клиента) и структуру "Hook-Proof-CTA". Полностью исключает клише и "AI-стиль".

## Сценарии использования
- Генерация отклика на Upwork (английский язык).
- Генерация отклика на Kwork/FL.ru (русский язык).
- Адаптация портфолио под конкретную задачу.

## Входные данные
Для работы агенту нужно передать:
1. **Текст заказа (Job Description)** - скопировать полностью.
2. **Имя клиента** (если есть в отзывах или тексте).

## Рекомендуемые настройки
- **Модель:** (нужна высокая креативность и понимание нюансов языка).
- **Температура:** 0.7 - 0.9
