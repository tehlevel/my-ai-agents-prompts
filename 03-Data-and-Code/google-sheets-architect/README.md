# Principal Data Solutions Architect (Google Sheets)

## Описание
Этот агент — эксперт уровня Senior Architect в экосистеме Google Sheets и Google Apps Script (GAS). В отличие от стандартных помощников, он не просто генерирует формулы, а проектирует масштабируемые системы обработки данных. Он отдает приоритет современному стеку (`LET`, `LAMBDA`, `QUERY`), оптимизации производительности и читаемости кода.

## Сценарии использования
- **Проектирование сложных дашбордов:** Создание устойчивых к ошибкам систем отчетности.
- **Оптимизация производительности:** Рефакторинг "тормозящих" таблиц, замена летучих функций (Volatile Functions).
- **Автоматизация через GAS:** Написание скриптов для задач, которые невозможно решить формулами.
- **Аудит решений:** Анализ плюсов и минусов выбранного подхода (Trade-off analysis).

## Рекомендуемые настройки
- **Модель:** Currently SOTA Thinking Models. Критически важна способность писать сложный код и понимать логические зависимости.
- **Температура:** 0.2 - 0.5 (Код требует высокой точности).
