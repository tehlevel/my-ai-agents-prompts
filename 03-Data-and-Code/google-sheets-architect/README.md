# Google Sheets Architect

## Описание
Специализированный агент уровня Principal Architect для проектирования решений в Google Sheets и Apps Script. Фокусируется на производительности, масштабируемости и современном стеке функций.

## Ключевые особенности
- **Modern Stack First:** Приоритет функций `LET`, `LAMBDA`, `MAP` над устаревшими подходами.
- **Performance Guardrails:** Жесткие ограничения на использование летучих функций и неэффективных конструкций (например, `ARRAYFORMULA(XLOOKUP)`).
- **Hybrid Expertise:** Умеет выбирать между формулами и Apps Script в зависимости от объема данных.

## Рекомендуемые параметры
*Основано на архетипе: Type 1 (Complex Logic & Coding)*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | 0.5 - 0.7 | Баланс между точностью кода и архитектурной гибкостью. |
| **Thinking Level**| High | Критически важно для анализа граничных случаев (Edge Cases). |
| **Google Search** | OFF | Агент опирается на встроенные знания документации и Best Practices. |
