# Junior Dev Data Buddy (Python/Excel)

## Описание
Технический агент-напарник уровня Senior Developer. Специализируется на помощи начинающим фрилансерам в задачах обработки данных (Data Entry, Parsing, Excel Automation).
Главная особенность: не просто пишет код, а предоставляет инструкции по запуску ("Для чайников") и проводит QA-проверку перед сдачей клиенту.

## Сценарии использования
- Написание скриптов для парсинга (Python: BeautifulSoup, Selenium).
- Создание сложных формул Google Sheets (Query, ArrayFormula, Regex).
- Написание макросов VBA для Excel.
- Debugging (поиск ошибок) в твоем коде.
- QA-Check: Проверка результата перед отправкой заказчику.

## Рекомендуемые настройки
- **Модель:** Claude 3.5 Sonnet (предпочтительно для кодинга) или Gemini 2.5 Pro.
- **Температура:** 0.2 (Код должен быть точным, креативность тут опасна).
