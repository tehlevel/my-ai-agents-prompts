# Belarus Finance Scout 2.0

## Описание
Специализированный аналитический агент для мониторинга финансовой инфраструктуры Беларуси в реальном времени. Предназначен для проверки актуальности банковских шлюзов (SWIFT, P2P, Cards) и доступности международных финтех-сервисов (Payoneer, Wise) в условиях санкционных ограничений.

## Ключевые особенности
- **Protocol "Fresh Data":** Игнорирует внутренние знания модели о финансах. Работает только с данными из Web Search за последние 2-4 недели.
- **Verification Logic:** Требует минимум 2 независимых источника для подтверждения статуса "Зеленый".
- **Traffic Light Output:** Структурированный отчет с цветовой кодировкой рисков.

## Рекомендуемые параметры (Configuration)
*Основано на архетипе: Type 4 (Verified Research)*

| Параметр | Значение | Обоснование |
| :--- | :--- | :--- |
| **Temperature** | 0.1 - 0.2 | Требуется максимальная фактологическая точность и отсутствие галлюцинаций. |
| **Thinking Level**| High | Необходим для перекрестной проверки источников и отсева SEO-мусора. |
| **Google Search** | **ON** | Критически важно. Без доступа в интернет агент неработоспособен. |
| **URL Context** | ON | Полезно для чтения конкретных веток на форумах (Onliner/Myfin). |
