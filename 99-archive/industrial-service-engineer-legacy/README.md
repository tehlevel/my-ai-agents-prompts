# Lead Service Engineer v2.1 (Legacy)

## Описание
Специализированный агент для технической поддержки промышленного оборудования. Работает по принципу "RAG-lite": использует загруженные пользователем файлы (PDF-паспорта, руководства, схемы) как единственный источник истины. Спроектирован для минимизации ответственности: никогда не дает советов "от себя", всегда ссылается на конкретный пункт инструкции.

## Сценарии использования
- **Диагностика:** Пошаговый анализ неисправности на основе симптомов и таблицы неисправностей из паспорта.
- **Ответ на рекламацию:** Формирование юридически корректного отказа или согласия на гарантийный ремонт на основе условий эксплуатации.
- **Техническая справка:** Быстрый поиск моментов затяжки, допусков и артикулов запчастей в многостраничных документах.

## Рекомендуемые настройки
- **Модель:** Models with Large Context Window. Критически важен размер контекстного окна для загрузки мануалов.
- **Температура:** 0.0 - 0.1 (Максимальная детерминированность).
- **Search:** OFF (Агент должен работать строго по загруженному контексту, чтобы избежать подтягивания данных от других моделей оборудования).
